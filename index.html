Peptide Toxicity Prediction using Ensemble Machine Learning (Super Learner Framework)

  • Developed a peptide toxicity classification pipeline using 438 sequence-derived physicochemical features, stratified cross-validation, and multiple ML classifiers for robust prediction
  • Applied XGBoost feature importance and Super Learner ensemble stacking, achieving 91.55% accuracy and 0.83 MCC on toxin vs non-toxin peptide prediction
